# ЁЯОм Netflix Clone тАУ Advanced Movie Streaming UI

A modern Netflix-inspired movie browsing web application built with **HTML, CSS, and JavaScript** using the **TMDB API**. This project allows users to explore trending, popular, top-rated, and upcoming movies, search for titles, filter by genre, view movie details, watch trailers, and maintain a personal watchlist.

---

## ЁЯУМ About The Project

This project was created to replicate the core browsing experience of Netflix while learning frontend development concepts such as:

- API Integration
- Dynamic DOM Manipulation
- Local Storage
- Responsive Web Design
- Search Functionality
- Infinite Scrolling
- Movie Recommendation Layouts

The application fetches real-time movie data from The Movie Database (TMDB) API and presents it in a clean and interactive interface.

## ЁЯЪА Getting Started

### 1. Clone The Repository

```bash
git clone https://github.com/yourusername/netflix-clone.git
```

### 2. Navigate To Project Folder

```bash
cd netflix-clone
```

### 3. Get TMDB API Key

Create a free account on TMDB and generate an API key.

### 4. Add Your API Key

Replace:

```javascript
const API_KEY = "PASTE_YOUR_REAL_TMDB_API_KEY";
```

with your actual TMDB API key.

### 5. Run The Project

Open `index.html` in your browser.
